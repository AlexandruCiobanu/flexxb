Packagecom.googlecode.flexxb.persistence
Interfacepublic interface IPersistable
ImplementorsPersistableObject, ReferenceList

Interface defining an object that can be persisted. It exposes three methods:
  • modified - check if object has been changed since its last save
  • commit - make changes done from the last save persistent
  • rollback - revert to the initial state of the object
