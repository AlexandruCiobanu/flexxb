Property detail
enableLoggingproperty
public var enableLogging:Boolean = false

Use this flag to control whether to permit logging or not. This is mostly intended for debugging the (de)serialization process.
Logging can slow down the process up to 20 times. Use with caution!

escapeSpecialCharsproperty 
public var escapeSpecialChars:Boolean = false

Set this flag to true if you want special chars to be escaped in the xml. If set to false, text containing special characters will be enveloped in a CDATA tag. This option applies to xml elements. For xml attributes special chars are automatically escaped.
